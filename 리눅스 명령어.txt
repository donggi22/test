기본 Linux 명령어와 설명
ls
설명: 현재 디렉토리에 있는 파일과 폴더 목록을 보여줍니다.
예시: ls -l (자세한 정보와 함께 목록 표시), ls -a (숨김 파일 포함).

cd
설명: 디렉토리를 이동할 때 사용합니다. 예를 들어, 특정 폴더로 들어가거나 상위 폴더로 이동할 수 있습니다.
예시: cd /home/user (특정 경로로 이동), cd .. (상위 디렉토리로 이동).

pwd
설명: 현재 작업 중인 디렉토리의 경로를 출력합니다. 내가 어디 있는지 확인할 때 유용합니다.
예시: pwd (현재 경로 표시).

mkdir
설명: 새 폴더를 만듭니다.
예시: mkdir 새폴더 (새폴더라는 이름의 폴더 생성).

rm
설명: 파일이나 폴더를 삭제합니다. 주의해서 사용해야 합니다!
예시: rm 파일.txt (파일 삭제), rm -r 폴더 (폴더와 그 안의 내용 삭제).

cp
설명: 파일이나 폴더를 복사합니다.
예시: cp 파일.txt /home/user/복사본.txt (파일 복사), cp -r 폴더 새폴더 (폴더 복사).

mv
설명: 파일이나 폴더를 이동하거나 이름을 변경합니다.
예시: mv 파일.txt /home/user/ (파일 이동), mv 파일.txt 새이름.txt (이름 변경).

cat
설명: 파일 내용을 터미널에 출력하거나 파일을 합칩니다.
예시: cat 파일.txt (파일 내용 보기), cat 파일1.txt 파일2.txt > 결과.txt (파일 합치기).

touch
설명: 빈 파일을 만들거나 기존 파일의 수정 시간을 업데이트합니다.
예시: touch 새파일.txt (새 파일 생성).

man
설명: 명령어의 사용법과 옵션을 알려주는 매뉴얼을 보여줍니다.
예시: man ls (ls 명령어의 매뉴얼 보기).

grep
설명: 파일이나 출력에서 특정 패턴을 검색합니다.
예시: grep "검색어" 파일.txt (파일에서 특정 단어 찾기).

chmod
설명: 파일이나 폴더의 권한을 변경합니다.
예시: chmod 755 파일.sh (파일에 실행 권한 추가).

ps
설명: 현재 실행 중인 프로세스를 보여줍니다.
예시: ps aux (모든 프로세스 정보 보기).

kill
설명: 특정 프로세스를 종료합니다. 프로세스 ID(PID)를 알아야 합니다.
예시: kill 1234 (PID 1234인 프로세스 종료).

df
설명: 디스크 사용량을 확인합니다.
예시: df -h (용량을 사람이 읽기 쉽게 표시).

top
설명: 시스템의 실시간 프로세스와 리소스 사용량을 보여줍니다.
예시: top (실시간 모니터링 시작).
sudo
설명: 관리자 권한으로 명령어를 실행합니다.
예시: sudo apt update (패키지 목록 업데이트).

apt (Debian/Ubuntu 계열)
설명: 소프트웨어 패키지를 설치, 업데이트, 제거합니다.
예시: sudo apt install vim (vim 설치).

find
설명: 파일이나 폴더를 검색합니다.
예시: find / -name "파일.txt" (전체 시스템에서 파일.txt 검색).

whoami
설명: 현재 로그인한 사용자의 이름을 보여줍니다.
예시: whoami (현재 사용자 확인).

추가 팁

명령어 조합: | (파이프)를 사용해 명령어를 연결할 수 있습니다. 예: ls | grep txt (txt로 끝나는 파일만 보기).
도움말: 명령어 뒤에 --help를 붙이면 간단한 사용법을 볼 수 있습니다. 예: ls --help.
탭 완성: 명령어나 파일 이름을 입력하다가 Tab 키를 누르면 자동 완성됩니다.
